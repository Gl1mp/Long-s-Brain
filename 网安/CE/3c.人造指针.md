我们有时候只是要临时修改一下值，只需造一个临时的基址就可以了，这就被称为人造指针/基址。本文以PVZ中的阳光修改为例。
## 自动汇编中的AOB
首先我们找到首地址，锁定汇编指令后反汇编，打开自动汇编工具。选择AOB注入，修改此处代码。
### AOB是什么？
**AOB** 是 Cheat Engine (CE) 中最核心、最常用的一个缩写，全称是 **Array of Byte**，有以下名字：
- **字节数组**
- **字节序列**
- **特征码**（最常见的俗称）
- **签名**（signature 的意思）
简单来说，AOB 就是**一串连续的机器码（十六进制字节）**，它代表了游戏/程序中某一段**特定的汇编指令**在内存里的真实样子。通过这串机器码，可以持续追踪这串汇编指令。
## 代码

  版本: 
  日期   : 2026-03-08
  作者 : Gl1mp
}

[ENABLE]

aobscanmodule(INJECT,PlantsVsZombies.exe,04 01 88 60 55 00 00) // should be unique
alloc(newmem,$1000)
alloc(mybase,$1000) //开辟mybase指针的空间

label(code)
label(return)

newmem:

code:
  mov [mybase],eax //将eax的值放入mybase指向的存储空间中
  add [eax+00005560],ecx
  jmp return

INJECT+01:
  jmp newmem
  nop
return:
registersymbol(INJECT)
registersymbol(mybase) //注册使得其能在CE中被识别

[DISABLE]

INJECT+01:
  db 01 88 60 55 00 00

unregistersymbol(INJECT)
unregistersymbol(mybase) //取消注册
dealloc(newmem)
dealloc(mybase) //释放内存
