## C
```C
int a;
int* p = &a; //一级指针
int** p= &p; //二级指针
```
简单来说，指针的级数就是他的星号的数量。
## CE
**基址>一级指针>二级指针>...>最终地址（首地址）**
举一个例子，在PVZ游戏中，我们直接通过数值的变化定位到了地址`0161E458`。这个地址所对应的内存空间中存放着阳光值。我们将这个地址称为首地址。
然后我们开始逆向，抓取到了命令：
```
10004910A - 89 46 18  - mov [rsi+18],eax
RSI=0161E440
```
RSI中的值`0161E440`就是一级指针了。接下来我们继续寻找，依次为二级指针、三级指针……最后找到了一个基址。
### 公式
$$目标地址=[[[基址] + 偏移1] + 偏移2] + … + 最后偏移$$
最后再将其解引用一次就找到了目标值。
