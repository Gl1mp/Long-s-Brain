## 基本示例
```CPP
#include <vector>
#include <iostream>
#include <algorithm>
int main()
{
	std::vector<int> values = {2, 4, 0, 2, 1};
	std::sort(values.begin(),values.end());
	for (int val : values)
	{
		std::cout << val << "\n";
	}
}
```
`sort`函数需要两个迭代器：
- begin()：返回指向**第一个元素**的迭代器。
- end()：返回指向**容器末尾之后**的迭代器（哨兵位置），用于判断遍历结束。
不仅如此，还有一个可选的第三个参数：比较器（Comparator）。如果没有提供，那么将默认使用`operator <`进行比较，对元素进行升序排序（从小到大）。
## 多种比较器
### fuctional
在导入了`functional`库后，可以使用：
```CPP
//std::sort(values.begin(),values.end(),std::greater<int>());
std::sort(values.begin(),values.end(),std::greater<>{}); // 更通用
```
### lambda
```CPP
std::sort(values.begin(),values.end(),[](int a, int b)
	{
		if (a == 0) return true;
		if (b == 0) return false;
		if (a > b)
		{
			return true;
		}
		else return false;
	});
```
我们通过给他一个匿名函数，有这样的特殊排序规则：
>把0放在最前面，其他的降序排序
