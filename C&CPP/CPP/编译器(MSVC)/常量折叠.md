**常量折叠（Constant Folding）** 是编译器在**编译阶段**（而不是运行时）对**常量表达式**进行求值，并直接用计算结果替换原表达式的优化技术。以减少运行时的计算开销，提高程序执行效率，同时让生成的代码更小、更快。
### 例
```CPP
const int a = 10;
const int b = 20;

// 编译器看到下面这行，会直接优化成：
int c = a + b * 3;          // → 编译后相当于 int c = 70;

// 更明显的例子：
int x = 5 + 8;              // 编译后直接变成 int x = 13;
int y = 100 - 37 * 2;       // 编译后直接变成 int y = 26;
double z = 3.14 * 2.0;      // 编译后直接变成 double z = 6.28;
```
甚至字符串字面量拼接（某些编译器支持有限的折叠）：
``` CPP
const char* s = "Hello" " " "World!";   // 很多编译器直接折叠成 "Hello World!"
```
