## 什么是链接属性
链接属性分为三种。**链接属性只在[[../文件作用域|文件作用域]]才有意义**。一旦进入块作用域，就没有链接属性了。

| 链接属性     | 含义            | 跨文件访问 |
| -------- | ------------- | ----- |
| **外部链接** | 整个程序共享，所有文件可见 | ✅ 可以  |
| **内部链接** | 仅当前文件可见       | ❌ 不能  |
| **无链接**  | 只是类型/别名，不占用存储 | 不适用   |
## 不同声明的默认链接属性
**在文件作用域下**，不同声明的默认链接属性如下：

|声明类型|默认链接属性|示例|
|---|---|---|
|**变量（非 const）**|外部链接|`int x;`|
|**变量（const）**|**内部链接**|`const int x = 1;`|
|**变量（constexpr）**|**内部链接**|`constexpr int x = 1;`|
|**函数**|外部链接|`void func();`|
|**类/结构体**|无链接（类型本身）|`class MyClass {};`|
|**枚举**|无链接（类型本身）|`enum Color { RED };`|
|**typedef/using**|无链接（别名本身）|`using Int = int;`|
## 注意点
### const 变量的特殊性

C++中文件作用域的 `const` 变量默认是内部链接，而C语言默认是外部链接。如果需要让`const`变量跨文件共享，必须显式加 `extern`。例如`extern const int a = 3;`而在外部链接时就使用`extern const int a;`。
### 函数默认是外部链接
如果想让函数只在本文件内使用，加 `static` 或放入匿名命名空间。
