## 基础使用
```CPP
#include <iostream>
class Entity
{
public:
	void Print()
	{
		std::cout << "hi!";
	}
};
int main()
{
	Entity e;
	e.Print();
	Entity* p = &e;
	*p.Print(); //TAG1
}
```
我们拥有一个简单的Entity类它可以输出一些东西。但是在此处TAG1的调用是错误的，因为.的优先级是会高于解引用运算符的。所以我们必须要加一个括号以改变优先级。
```CPP
(*p).Print(); //TAG1
```
当然更简单的是我们可以替换为`p->Print();`也就是我们的*箭头运算符*了。
## 运算符重载
```CPP
#include <iostream>

class Entity
{
public:
	void Print()
	{
		std::cout << "hi!";
	}
};

class ScopedPtr
{
public: // TAG2
	Entity* m_ptr;
public:
	ScopedPtr(Entity* entity)
		: m_ptr(entity){}
	Entity* GetPtr() const
	{
		return m_ptr;
	}	
	~ScopedPtr()
	{
		delete m_ptr;
	}
};
int main()
{
	ScopedPtr entity = new Entity{};
	entity.m_ptr->Print();
	entity.GetPtr()->Print();
}
```
我们有一个手动的作用域指针，他在离开作用域后会自动销毁堆上的内容。现在我们想要通过指针`entity`来访问`Entity`的成员函数`Print()`。我们有两种简单的方式：
- 将`TAG2`处设为`public`
- 新建返回地址的智能指针类中的方法`GetPrt()`。
