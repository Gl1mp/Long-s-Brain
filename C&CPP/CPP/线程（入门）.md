```CPP
#include <iostream>
#include <thread>
#include <chrono>

static bool s_finished = false;

void DoWork(){
	using namespace std::literals::chrono_literals;
	std::cout << std::this_thread::get_id() << "\n";
	while (!s_finished)
	{
		std::cout << "working...\n";
		std::this_thread::sleep_for(1s);
	}
}
int main()
{
	std::thread worker(DoWork);

	std::cin.get();
	s_finished = true;

	std::cout << std::this_thread::get_id() << "\n";

	worker.join();
}
```
这是一个例子，我们有一个*主线程* 和 一个*工作线程*。工作线程负责一直输出`working`直到主线程中得到了一个回车，使得`bool`变量变为了`true`后，停止其循环。有如下几个要点：
- `worker.join()`的意思是主线程需要等待工作线程完成之后才会继续进行
- `std::literals::chrono_literals;`可以进行时间的转换，例如这里的`1s`
- 我们可以通过`std::this_thread::get_id()`来获取当前线程的ID号
