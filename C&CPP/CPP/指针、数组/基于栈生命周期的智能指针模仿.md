```CPP
#include <iostream>

class Entity
{
public:
	Entity()
	{
		std::cout << "hi";
	}
	~Entity()
	{
		std::cout << "bye";
	}
};

class ScopedPtr
{
private:
	Entity* m_ptr;
public:
	ScopedPtr(Entity* ptr) 
		: m_ptr(ptr) {}

	~ScopedPtr()
	{
		delete m_ptr;
	}
};

int main()
{
	{
		ScopedPtr e = new Entity();
	}
}
```
这段代码由三个部分组成，主函数、智能指针类还有实体类。这个智能指针的设计思路非常简单，我们直接讲解代码原理。
1. 进入花括号作用域
2. `new Entity()`调用默认构造函数，在堆上创建`Entity`对象
3. 发生隐式转换，调用ScopedPtr的有参构造
4. 生成了一个对象，保存到了成员指针变量中
5. 当离开作用域时，调用析构函数，执行`delete m_ptr`释放内存
