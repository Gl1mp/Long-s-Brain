我们在[[../C/笔记库/数据类型与内存表示/联合（共用体）|联合（共用体）]]中已经提到了联合。此处给出一个CPP背景下的联合例子：
```CPP
#include <iostream>

struct vector2
{
	float x,y;
};

struct vector4
{
	union
	{
		struct 
		{
			float x, y, z, w;
		};
		struct
		{
			vector2 a, b;
		};
	};
	
};

void PrintVector2(const vector2& vec)
{
	std::cout << vec.x << "," << vec.y << std::endl;
}

int main()
{
	vector4 vector{12.0f, 13.0f, 19.0f, 9.0f};
	PrintVector2(vector.a);
	PrintVector2(vector.b);
	vector.z = 500.0f;
	PrintVector2(vector.a);
	PrintVector2(vector.b);
}
```
我们得到结果：
```CPP
12,13
19,9
12,13
500,9
```
很明显我们修改z的时候也修改了b中的第一个元素的内容，因为他们俩本身共用一个内存地址中 的值。只是他们的组织方式不一样而已。
