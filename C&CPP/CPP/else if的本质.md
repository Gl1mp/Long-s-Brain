在CPP中，`else if`不是一个独立的关键字，而是else和if的组合。
```CPP
if (条件1) {
    // 条件1为真时执行
}
else if (条件2) {  // 这实际上是：else { if (条件2) { ... } }
    // 条件2为真时执行
}
else {
    // 所有条件都不满足时执行
}
```
实际上应该是这样的的：
```CPP
if (条件1) {
    // 条件1为真
}
else {
    if (条件2) {  // else 后面跟了一个完整的 if 语句
        // 条件2为真
    }
    else {
        // 都不满足
    }
}
```
如果在一个程序中大量使用if语句，实际上会减缓程序的速度。因为在反汇编下我们能看到有大量的地址跳跃。
