```CPP
#include <iostream>
class People
{
public:
	int m_x;
public:
	People(int x)
		: m_x(x) {}
};
int main()
{
	People p1 = 3;
}

```
对于主函数中的这一句`People p1 = 3;`，发生了一个*隐式转换*。在CPP中，编译器最多允许一次隐式转换。他有这么几个步骤：
1. `People p1 = 3;`
2. `People p1 = People(3);`
3. `People p1(3);`
其中2和3会任意执行一句，因为编译器可能会优化拷贝构造。
