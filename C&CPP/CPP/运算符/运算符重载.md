```CPP
struct Vector2
{
	float m_x, m_y;

	Vector2(const float x, const float y)
		: m_x(x), m_y(y) {}

	Vector2 Add(const Vector2& other) const
	{
		return Vector2(m_x + other.m_x, m_y + other.m_y);
	}

	Vector2 operator+(const Vector2& other) const
	{
		return Add(other);
	}

	static void Print(const Vector2& vector)
	{
		std::cout << "x:" << vector.m_x << " y:" << vector.m_y << "\n";
	}
};
```
在此处我们有一个结构体（结构体除了可见性之外和类是基本完全一致的，是C语言的遗留产物）。
## 加号的重载
在此处定义了一个二维向量结构，并有一个成员方法`Add`。我们还可以进行运算符重载，也就是将加号赋予新的意义。
```CPP
Vector2 operator+(const Vector2& other) const
	{
		return Add(other);
	}
```
这段就是重载的关键代码。如果需要运算符重载，我们推荐同时编写成员方法和等价的重载规则。两者的实现方式一致，只是调用方式不同。可以实现一然后另外一个调用已实现的部分的就可以了。
**我们发现，明明是个二元运算符，为什么参数读进来只有一个呢？实际上是转换成了这样子的：`v1.operator+(v2)`。**这下子就很明朗了。
## <<的重载（输出重载
```CPP
std::ostream& operator<<(std::ostream& stream, const Vector2& other)
{
	stream << other.m_x << ", " << other.m_y;
	return stream;
}
```
我们通过引用的方式引用了输出流。并最后返回了`ostream`类型的`stream`。
## 等于和不等于的重载
```CPP
bool operator==(const Vector2& other) const
{
	if (other.m_x == m_x && other.m_y == m_y)
	{
		return true;
	}
	else return false;
}
bool operator!=(const Vector2& other) const
{
	return !(other == *this);
}
```
