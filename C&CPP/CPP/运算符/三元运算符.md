这里介绍三元运算符的一个推广形式，尽管他非常少被使用。
```CPP
#include <iostream>
using String = std::string;
using std::cout;
int main()
{
	constexpr int score = 60;
	String level = score > 80 ? score >= 60 ? "A" : "B" : "C";
	cout << level;
}
```
如果这么做你会发现输出了B，因为实际上三元运算符是右结合的。也就是说实际上是`String level = score > 80 ? (score >= 60 ? "A" : "B") : "C";`。自然会看到错误的逻辑结果。我们只需要加上一个括号`String level = score > 80 ? "A" : (score >= 60 ? "B" : "C");`。**或者我们根本就不该使用三元运算符嵌套**。
