## 涵盖多种情况的样例
```CPP
#include <iostream>
using String = std::string;
using std::cout;
class People
{
public:
	int m_num;
public:
	People()
		: m_num(0) {}
	People(int num)
		: m_num(num) {}
	void Print() const
	{
		cout << m_num << "\n";
	}
};
int main()
{
    cout << "--- 开始创建对象 ---\n";

    People p1;                 // 默认构造
    People p2(2);              // 参数构造
    People p3 = People(3);     // 参数构造
    People p4 = p2;            // 拷贝构造
    // 如果是 People p4; p4 = p2; 那么就对应的是赋值而非拷贝构造
    People p5 = 4;             // 隐式转换：参数构造
    People* p6 = new People(9); // 参数构造（堆上）
    People p7{ 90 };             // 统一初始化：参数构造
    People* p8 = new People[10]{}; // 10次默认构造（堆上数组）
	//People p9();    // ⚠️这是函数声明：声明一个名为p9、返回People类型、不接受参数的函数
	People p10{};   // 使用统一初始化（C++11），创建People对象并进行值初始化

    cout << "\n--- 输出值 ---\n";
    p1.Print();
    p2.Print();
    p3.Print();
    p4.Print();
    p5.Print();
    p6->Print();
    p7.Print();
    // 数组中的元素未初始化，需要循环打印...

    cout << "\n--- 清理内存 ---\n";
    delete p6;                  // 析构 9
    delete[] p8;                // 析构数组中的10个对象

    cout << "--- 程序结束，栈对象自动析构 ---\n";
    // p1-p7 在这里自动析构
}
```
## 若干种典型情况
### 默认初始化和值初始化
| 写法          | 初始化类型     | 默认值处理                                 |
| ----------- | --------- | ------------------------------------- |
| `Test a;`   | **默认初始化** | 成员变量可能未初始化（保留随机值）                     |
| `Test a{};` | **值初始化**  | 成员变量被零初始化（`int` → 0，`指针` → `nullptr`） |
### 聚合初始化
`test a = {1, 2, 3};`能够运行是因为test是聚合类型。
#### 聚合类型的条件
```CPP
class test {
public:
    int x;
    int y;
    int z;
    // 没有用户自定义的构造函数
    // 没有 private/protected 的非静态成员
    // 没有虚函数
    // 没有基类
};
```
满足以上条件时，可以用 `{}` 按顺序初始化成员。
#### 例子
```CPP
class test {
public:
    int x;
    int y;
    int z;
};

int main() {
    test a = {1, 2, 3};    // ✅ 聚合初始化，a.x=1, a.y=2, a.z=3
    test b {4, 5, 6};      // ✅ C++11 起可以省略 =
    test c = {1, 2};       // ✅ 部分初始化，c.z=0（零初始化）
    test d = {};           // ✅ 所有成员初始化为 0
    
    // test e = {1, 2, 3, 4};  // ❌ 错误：太多初始化值
    
    return 0;
}
```
## 堆栈中创建对象
### 栈中创建对象
除了`p6`和`p8`，其他的都是栈上的对象。他们的生命周期由程序自动控制。但是如果较大或者数量较多，容易发生栈溢出，此时可以考虑在堆上生成。
### 堆中创建对象
`p6`和`p8`是在堆上的对象。内存中的堆区较大，适合存放较多或较大的对象。他们的生命周期很长，需要手动释放，否则容易发生内存溢出。请注意，`new`返回的是一个指针，需要存在指针变量中。
