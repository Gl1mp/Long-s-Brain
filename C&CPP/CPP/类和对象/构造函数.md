## 定义
### 构造函数
构造函数是类中用于初始化对象的特殊成员函数，可以有多个（重载）。
### 默认构造函数
默认构造函数是**不需要参数就能调用的构造函数**，包括：
- 无参构造函数
- 所有参数都有默认值的构造函数
## 默认构造&有参构造
- 如果没有任何构造函数，编译器就会自动生成一个默认构造函数。如果初始化对象的时候加上了圆括号或花括号，就会初始化所有值为0，否则会进行"默认初始化"（垃圾值）。具体实现和编译器有关系
- 如果有带参的构造函数但是没有默认构造函数，调用默认构造的话就会报错
- 如果有例如`People() = delete;`的明确禁用默认构造，当你调用默认构造函数的时候就会发生错误
- `People() = default;`我们可以在类中加上这句以强制生成默认构造
### 未显示定义构造函数
```CPP
#include <iostream>

class People
{
public:
	int m_x;
	int m_y;
};
int main()
{
	auto* p1 = new People;
	std::cout << p1->m_x << p1->m_y;
	delete p1;
}
```
这样输出的结果是随机值（垃圾值），因为执行的是默认初始化。我们需要将其变成`auto* p1 = new People{};`，这样就执行了值初始化，也就是说都变成了0。
## 禁止类外实例化（通过更改构造函数）
```CPP
People() = delete;
```
这样会使People()没有默认构造函数。
```CPP
private:
	People() {
		m_x = 0;
		m_y = 0;
	}
```
或者我们让默认构造函数变成一个私有函数，外部无法访问。
