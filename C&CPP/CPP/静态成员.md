静态成员属于类本身的成员，不属于某个具体的对象。
```CPP
class People {
    static int count;  // 静态成员
    int age;           // 非静态成员（普通成员）
};
```
下面这是一个典型的例子。
```CPP
#include <iostream>
class People {
private:
	static int m_peo; // 声明，非const的静态成员变量不能在类内直接初始化
	static const int ID = 0xa34f12e; // 声明+定义，要在类内完成初始化
public:
	static void ShowPeople() {
		std::cout << m_peo << std::endl; // 静态成员函数可以直接访问静态成员变量
	}
};
int People::m_peo = 5; // 定义，非const的静态成员变量应在类外初始化

int main() {
	People::ShowPeople(); // 通过类名直接调用静态成员函数
	return 0;
}
```
## 总结
- **静态函数**：只能访问静态成员（变量和函数）
- **非静态函数**：可以访问所有成员（静态和非静态）
其实理解起来非常简单，除了静态成员不能访问非静态之外，其他的都是可以的。
