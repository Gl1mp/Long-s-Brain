涉及到两个核心函数`fopen()`和`fclose()`。

### 初识
文件大体分两类：程序文件和数据文件。程序文件有常见的源文件(.c)，目标文件(.obj)，可执行程序(.exe)。数据文件就很宽泛了。
## 文件指针
每一个被使用的文件，都在内存中有一个对应的文件信息区，里面包含了多种参数。这些信息存储在结构体变量中，他的类型名位`FILE`，是一个`typedef`后的名字。那么对应的指针类型就为`FILE*`。我们通过这个指针来维护这个结构体变量。
## fopen()
```C
FILE * fopen ( const char * filename, const char * mode );
```
只要注意他的返回值是一个文件指针类型就行了。
## fclose()
```C
int fclose ( FILE * stream );
```
### 返回值说明
>If the stream is successfully closed, a zero value is returned.  
On failure, [EOF](https://cplusplus.com/EOF) is returned.
