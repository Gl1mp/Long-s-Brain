## 定义
结构是一些值的集合，这些值称为成员变量。结构的每个成员可以是不同类型的变量。每一个结构体都是独立的。
### 样例
```C
#include <Stdio.h>
struct Man {
	char name[20];
	int age;
}man1 = {"Nick", 18}, man2;
int main()
{
	printf("%d", man1.age);
}
```
上方的变量`man1`，`man2`都是全局变量。尽量不要这么做。
我们可以用`typedef`来简化这个过程。
```C
#include <Stdio.h>
typedef struct Man {
	char name[20];
	int age;
}Man;
int main()
{
	Man man1 = {"Nick", 18};
	printf("%d", man1.age);
}
```
## 匿名结构体
我们创建结构体的时候可以不加名字，也就是说在此处要创建好结构体变量了，否则就无法再使用了。
```C
#include <Stdio.h>
struct {
	char name[20];
	int age;
}man1 = { "Nick", 18 };
int main()
{
	printf("%d", man1.age);
}
```
## 自引用
以链表为例子引入，我们需要在一个结构体节点中放入本节点的值和下一个节点的地址。
```C
struct Node {
	int value;
	struct Node* next;
};
```
这样就实现了自引用。使用`typedef`可以简化。
```C
typedef struct Node { int value; Node* next; }Node;
```
我们进行这样的简化，可是发现报错了。因为 `Node` 这个类型名在结构体定义**完成后**才被 `typedef` 定义。
```C
typedef struct Node {
    int value;
    struct Node* next;  // 这里还需要用 struct Node*
}* p;  // p 是 struct Node* 的别名
```
这样的实现就是正确的啦！我们还有一种实现方式：
```C
typedef struct Node {
	int value;
	Node* next;
}Node;
typedef struct Node* p;
```
这样也是正确的也是推荐的。
## 结构体里放结构体
```C
#include <Stdio.h>
typedef struct Score {
	int Chinese;
	int Math;
}Score;
typedef struct Man {
	char name[20];
	int age;
	struct score s;
}Man;
int main()
{
	Man A = { "Nick", 18, {91, 78}};
	printf("%d", A.s.Chinese);
}
```
- Score和Man首字母尽量大写，说明这是一个类型
- 要注意我们访问的是变量，不是类型
