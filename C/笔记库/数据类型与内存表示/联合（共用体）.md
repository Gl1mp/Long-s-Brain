联合的空间是共用的。他的大小至少为最大成员的大小。
## 通过联合来判断大小端
```C
int check_sys(){
	union {
		int a;
		char b;
	}u;
	u.a = 1;
	return u.b;
}
```
其实和[[大小端问题#指针转换实现大小端判断]]高度类似。
## 判断存储大小
他的存储大小判断主要取决于两个因素：
1. 最大成员变量的大小
2. 最大成员的对齐数
```C
union Example {
    int a;          // 4字节（假设int是4字节）
    char b[10];     // 10字节
};
```
- `int a`的对齐数为4
- `char b[10]`的对齐数为1

所以取最大的对齐数为4。而最大的成员变量大小为10，总大小必定为4的倍数，即为12。
