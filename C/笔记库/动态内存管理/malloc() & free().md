## malloc()
### 函数原型
```C
void* malloc (size_t size);
```
- `size`是要开辟连续内存的大小
- 返回一个指针指向开辟的空间，如果开辟失败，那么返回空指针（需要空指针检查）
- 注意返回值是`void*`，使用时需要强制类型转换
## free()
动态内存空间被开辟后，必须要释放，不然会造成内存泄露。**内存泄露是指程序分配了内存但忘记释放，导致可用内存逐渐减少的现象。**
### 函数原型
```C
void free (void* ptr);
```
- `ptr`是指向动态内存空间的指针
- **必须要指向是动态的内存空间的指针**
- 无返回
- 如果释放NULL指针，那么不会进行任何操作
- 释放`ptr`后，并不会修改这个指针，导致其变成了**野指针**，我们必须要将其变为空指针。
## 样例
```C
#include <stdio.h>
#include <stdlib.h>
int main() {
	int *p = (int*)malloc(5 * sizeof(int));
	if(!p) return;
	int i = 0;
	for (i = 0; i < 5; i++) {
		*(p + i) = i;
	}
	for (i = 0; i < 5; i++) {
		printf("%d ", * (p + i));
	}
	free(p);
	p = NULL;
	return 0;
}
```
```C
0 1 2 3 4
```
- 不要忘记返回值类型转换
- 一定要记得释放
- `p`释放后要置为空指针
- 有`return 0`的代码更加标准
- 检查是否返回空指针
