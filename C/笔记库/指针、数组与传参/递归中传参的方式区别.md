
| 写法                                    | 推荐度   | 说明           |
| ------------------------------------- | ----- | ------------ |
| `return func(n + 1)`                  | ⭐⭐⭐⭐⭐ | 清晰，无副作用      |
| `int next = n + 1; return func(next)` | ⭐⭐⭐⭐  | 非常明确         |
| `return func(++n)`                    | ⭐⭐    | 能用但容易混淆      |
| `return n + func(++n)`                | ⭐     | 危险！计算顺序依赖编译器 |
