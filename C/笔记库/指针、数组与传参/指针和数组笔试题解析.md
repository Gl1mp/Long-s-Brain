## 综合应用
### 1
```C
#include <stdio.h>
int main()
{
	int a[5] = { 1, 2, 3, 4, 5 };
	int *ptr = (int *)(&a + 1);
	printf( "%d,%d", *(a + 1), *(ptr - 1));
	return 0;
}
```
```C
2,5
```
### 2
```C
//由于还没学习结构体，这里告知结构体的大小是20个字节
#include <stdio.h>
struct Test
{
	int Num;
	char *pcName;
	short sDate;
	char cha[2];
	short sBa[4];
}*p;
//假设p 的值为0x100000。 如下表表达式的值分别为多少？
//已知，结构体Test类型的变量大小是20个字节
int main()
{
	printf("%p\n", p + 0x1);
	printf("%p\n", (unsigned long)p + 0x1);
	printf("%p\n", (unsigned int*)p + 0x1);
	return 0;
}
```
```C
0x100014

```
%p会将指针的地址以十六进制形式打印出来。
1. 指针加法，因为0x1对应的十进制下就是1，地址前进20字节。也就是`0x100014`了
2. 将 p 转换为`unsigned long`类型，这是数值加法，直接加 1，即 0x100000 + 1 = 0x100001。
3. 将 p 转换为`unsigned int*`类型指针，`unsigned int`大小通常为 4 字节。指针加法基于所指类型的大小，因此加 1 相当于地址前进 4 字节，即 0x100000 + 4 = 0x100004。
### 3
```C
#include <stdio.h>
int main()
{
	int a[4] = { 1, 2, 3, 4 };
	int *ptr1 = (int *)(&a + 1);
	int *ptr2 = (int *)((int)a + 1);
	printf( "%x,%x", ptr1[-1], *ptr2);
	return 0;
}
```
```C
4,2000000
```
这里的问题主要是在ptr1上。这是a被转化为int类型，在加法的时候只前进了一个字节。从此开始的连续四个字节被解释为一个整数。这里还有小端存储的问题
```C
01 00 00 00 02 00 00 00
   |         | 
   从此处开始  到此结束
```
然后因为有小端存储，翻转过来。以`%d`打印的时候，按照十六进制无符号整数打印。
### 4
```C
#include <stdio.h>
int main()
{
	int a[3][2] = { (0, 1), (2, 3), (4, 5) };
	int *p;
	p = a[0];
	printf( "%d", p[0]);
	return 0;
}
```
这个括号中其实是一个逗号表达式，值为最后一项。也就是说，显示初始化的有：
```C
{1, 3, 5}
```

|      | col0 | col1 |
| ---- | ---- | ---- |
| row0 | 1    | 3    |
| row1 | 5    | 0    |
| row2 | 0    | 0    |
| ---- | ---- | ---- |
后面的分析较为简单，`a[0]`为二维数组的第一个元素，也就是第一行这一个一维数组。那么p就是一维数组的第一个元素。
```C
1
```
### 5
```C
#include <stdio.h>
int main()
{
	int a[5][5];
	int(*p)[4];
	p = a;
	printf( "%p,%d\n", &p[4][2] - &a[4][2], &p[4][2] - &a[4][2]);
	return 0;
}
```
```C
FFFFFFFC,-4
```
这道题的问题主要是就是a和p的类型不一致。同为数组指针，一个会前进`5x4=20`字节，另一个会前进`4x4=16`字节。
- `&p[4][2]`计算方式就是：`起始地址+4*4*4+2*4=a+72`
- `&a[4][2]`计算方式就是：`起始地址+4*5*4+2*4=a+88`
相减的结果为`-16/sizeof(int)=-4`，-4的补码为`11111111 11111111 11111111 11111100`,以无符号形式打印就是`FFFFFFFC`啦！

### 7
```C
#include <stdio.h>
int main()
{
	char *a[] = {"work","at","alibaba"};
	char**pa = a;
	pa++;
	printf("%s\n", *pa);
	return 0;
}
```
```C
at
```
a是一个指针数组，每个元素里放着指向char的指针。pa是指向指针数组第一个元素的指针。pa++就意味着他指向了at，也就是说会输出at。
### 8
```C
#include <stdio.h>
int main()
{
char *c[] = {"ENTER","NEW","POINT","FIRST"};

char***cpp = cp;

printf("%s\n", *cpp[-2]+3);
printf("%s\n", cpp[-1][-1]+1);
return 0;
}
```
