## void* 简介

void* 是 C 语言中一种特殊的指针类型，也称为“泛型指针”或“虚空指针”。它可以指向任何数据类型的对象，但不能直接访问或操作所指向的数据。void* 常用于实现泛型编程、函数参数传递和内存管理。
### 基本特性

- **声明方式**：void* ptr;。这里的 ptr 可以存储任何类型的地址，而不指定指向的具体类型。
- **灵活性**：void* 可以被赋值给其他指针类型，反之亦然（隐式转换）。例如：
```C
int num = 42;
void* vp = &num;  // 合法，&num 是 int*，隐式转为 void*
int* ip = vp;     // 合法，void* 隐式转为 int*
```

- **限制**：不能直接解引用（_vp）或进行指针算术（vp + 1），因为 void_ 不知道元素大小。必须先转换为具体类型：

```C
printf("%d\n", *(int*)vp);  // 先转为 int*，再解引用
```
