我们知道文件在打开的时候会有文件指针，我们可以通过文件指针来访问文件中的内容。主要有`fseek()`，`ftell()`，`rewind()`这三个函数。
## rewind()
让文件指针归位。`rewind(p)`即可。
## ftell()
告诉你当前文件指针相对于初始位置的偏移量。
```C
long int ftell ( FILE * stream );
```
这就是函数原型，直接使用即可。
## fseek()
```C
int fseek ( FILE * stream, long int offset, int origin );
```
这就是函数原型，让文件指针到origin为起点，偏移量为offset的地方去。
