```C++
#include <iostream>
using namespace std;
int& Hello() {
	static int a = 3;
	return a;
}
int main() {
	int &g = Hello();
	cout << g << endl;
	g = 7;
	cout << g << endl;
	Hello() = 9;
	cout << g << endl;
	return 0;
}
```
## 常量引用
```C++
#include <iostream>
using namespace std;
void print(const int& a) {
	cout << "a:" << a << endl;
}
int main() {
	int a = 3;
	print(a);
	print(9);
}
```
常量引用就是用const修饰int&使得这个引用变量无法被修改。如果有`const int& a = 3`这也将是合法的，因为编译器会开一块只读的temp存放3，然后a就指向这个temp。
