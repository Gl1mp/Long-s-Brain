## 指针类型
### 字符指针
- 类型：`char*`
- 指向的是字符类型的元素

#### 两个例子
1. ```C
   char ch = 's';
   char* p = &ch;
   ```
   p存放的是ch变量的地址。
2. ```C
   const char* p = "Hi!";
   ```
   - p存放的是字符串首元素的地址
   - const修饰char*，意为无法通过p来修改p指向的元素，但是可以修改p自身
   - "Hi!"这个字符串放在只读区中，无法修改
#### 试题
```C
#include <stdio.h>
int main()
{
	char str1[] = "hello bit.";
	char str2[] = "hello bit.";
	const char *str3 = "hello bit.";
	const char *str4 = "hello bit.";
	if(str1 ==str2)
		printf("str1 and str2 are same\n");
	else
		printf("str1 and str2 are not same\n");
	if(str3 ==str4)
		printf("str3 and str4 are same\n");
	else
		printf("str3 and str4 are not same\n");
	return 0;
}
```
- str1和str2不一样：因为他们两个是独立的两个字符数组，首元素地址不同
- str3和str4一样：因为他们都指向只读区的同一个字符串`hello bit.`的首元素
### 整型指针
- 类型：int*`
- 指向的是整型类型的元素
### 数组指针
- 类型：int (\*)\[size\]`
- 指向的是整型类型的元素
#### 说明
给出`int (*p)[10]`：
- \*和p结合，说明p是一个指针
- 指向\[10]这个数组
- 数组里面的元素类型是int
**注意**：\[]优先级比\*高，必须要加括号
#### 区分
1. `int *p1[10];`指针数组
2. `int (*p2)[10];`数组指针
#### 例子
`int arr[] = {1, 2};`
如果我们此时&arr，那么取出来的是整个数组的地址。应该用数组指针来存放。在指针加法中，如果是数组指针+1，那么会跳过整个数组。更多可以参看：[[arr和&arr]]

## 数组类型
### 指针数组
- 类型：`type *[size]`
- 例子：int \*\[5\]
- 本质是一个数组，数组里每个元素都是指针
#### 区分
- 整形指针的数组`int* arr1[10];`
- 一级字符指针的数组`char *arr2[4];`
- 二级字符指针的数组`char **arr3[5];`
#### 函数指针数组
如果指针数组中存的函数的地址，那么就是函数指针数组了。
参见：[[指针视角下的函数#函数指针数组]]
## 两种类型的辨析
```C
int arr[5];
int *parr1[10];
int (*parr2)[10];
int (*parr3[10])[5];
```
1. 数组——有五个元素，元素类型为int
2. 指针数组——有十个元素，元素类型为\*int
3. 数组指针——指向一个有十个元素，元素类型为int的数组
4. 数组指针数组——存放数组指针的数组，数组有十个元素，存放着指向一个有五个元素，元素类型为int的数组
