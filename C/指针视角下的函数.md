- 类型：`type (*)(paratype1,paratype2...)`
- 指向的是一个返回值为type的函数
- 这个括号不能省，否则就成返回type\*的函数了
- e.g. `int (*ptr)(char, int);`
## 函数指针的使用
我们知道所有函数都有自己的地址，假设有函数Add:
```C
#include <stdio.h>
int add(int a,int b) {
	return a + b;
}
int main() {
	int (*pAdd)(int, int) = &add;
	//int (*pAdd)(int, int) = add;
	int result = pAdd(10, 20);
	//int result = (*pAdd)(10, 20);
	printf("result=%d\n", result);
}
```
- `add`和`&add`的效果是一致的
- `*pAdd`和`pAdd`的效果是一致的
- **注意**：`*pAdd`一定要加括号
## 两个有趣的例子
### 例1
   ```C
(*(void (*)())0)();
```
`void (*)()`是函数指针类型。
```C
(*(type)0)();
```
意思就是先将0强制类型转换为函数指针，接下来再解引用使用这个函数，后面的参数为空。
本质上的形式就是`*p()`，类似于上面[[#函数指针的使用]]中的例子。
### 例2

```C
void (*signal(int, void(*)(int)))(int);
```
我们观察这个式子，最外面有个`void`，似乎是声明的形式。
```C
*signal(int, void(*)(int))
```
signal是个函数，他有两个参数`int`和`void(*)(int)`。去掉这些，我们得到了：
```C
void (*)(int);
```
这就是一个函数指针类型，说明这个signal的返回值是一个函数指针类型。
**所以**，这句话是一个函数声明。说明signal函数返回的是函数指针，有两个上述所说类型的参数。
#### 简化
我们利用typedef来简化：
```C
typedef void (*sighandler_t)(int);
sighandler_t signal(int, sighandler_t);

```

## 函数指针数组
我们可以把很多函数的地址放在一个数组中，这样就是一个函数指针数组了。
```C
int *arr[5]
```
这意味着数组中的每个元素都是int*。我们接着继续改造：
```C
int (*arr[5])(int, char);
```
1. `arr`和`[5]`结合，说明arr是一个数组，有五个元素
2. `int (*)(int, char)`说明了每个元素的类型
### 用途
我们可以用函数指针数组来当**转移表**。我们先完成加减乘除运算函数的编写。然后：
```C
int (*arr[4])(int, int) = {Add, Sub, Mul, Div};
```
建立一个转移表，将四个函数的地址放入其中。
```C
int ret = (*arr[input])(x, y);
```
我们只需要输入`x`和`y`就可以了。
## 回调函数
>**回调函数**就是一个通过函数指针调用的函数。如果你把函数的指针（地址）作为参数传递给另一个函数，当这个指针被用来调用其所指向的函数时，我们就说这是回调函数。回调函数不是由该函数的实现方直接调用，而是在特定的事件或条件发生时由另外的一方调用的，用于对该事件或条件进行响应。

通俗的说法就是将函数地址当作参数传递，在必要时刻通过地址进行函数的使用。
