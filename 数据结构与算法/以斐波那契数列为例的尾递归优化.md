## 原始递归版本
这个版本没有进行任何的优化，是公式：
$$ Fib(n) = Fib(n-1)+Fib(n-2) (n>=2)$$
$$ Fib(0)=0\ Fib(1)=1$$
的直观表达。
```C
#include <stdio.h>
int fib(int n) {
	if(n == 0) return 0;
	if(n == 1) return 1;
	return fib(n - 1) + fib(n - 2);
}
int main(){
	int n;
	scanf_s("%d", &n);
	int num = fib(n);
	printf("Fib:%d", num);
}
```
如果这么执行下去，那么产生了一个层数为n的递归树。
## 尾递归优化版本
**如果函数在返回前的最后一步才进行递归调用**，则该函数可以被编译器或解释器优化，使其在空间效率上与迭代相当。这种情况被称为尾递归（tail recursion）。
```C
#include <stdio.h>
int fib(int n, int a, int b) {
	if (n == 0) return a;
	return fib(n - 1, b, a + b);
}
int main(){
	int n;
	scanf_s("%d", &n);
	int num = fib(n, 0, 1);
	printf("Fib:%d", num);
}
```
