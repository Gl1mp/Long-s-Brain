| 阶段     | 工具                          | 输入文件           | 主要输出文件（中间文件）                        | 备注                          |
| ------ | --------------------------- | -------------- | ----------------------------------- | --------------------------- |
| 1. 预处理 | cl.exe (/P)                 | .cpp / .c / .h | **.i**                              | 展开宏、包含头文件、条件编译后纯文本          |
| 2. 编译  | cl.exe                      | .i（或直接 .cpp）   | **.obj**（默认不输出.asm） 可选 .asm（用 /FAs） | 把 C++ 转成机器码，生成 COFF 格式 .obj |
| 3. 汇编  | cl.exe 内置 或 ml.exe/ml64.exe | .asm（如果有）      | **.obj**                            | MSVC 通常不单独输出 .asm 再汇编       |
| 4. 链接  | link.exe                    | .obj + .lib    | **.exe** / **.dll** / **.lib**      | 合并所有 .obj，解析符号，生成可执行文件或库    |
